Bu implementasyonlar Windows ortmında Visual Studio ide ve Microsoft C++ Compiler (MSVC) derleyicisi kullanılarak geliştirilmiştir.

- Proje'de solution properties üzerinden hangi uygulamanın compile ve run edilebileceğini seçebilirsiniz.

- Metin düzenleyici projesi için kullanıcıdan dosya kaydetmeye ek olarak dosya okuma işlemleri de implemente edilmiştir."Metinler.txt" dosyasından veri okunabilir veya dosya kaydetme işleminde aynı dosya ismini girerek override şeklinde veri yazılabilir
- Metin şifreleme projesi için okunacak veriler "metin.txt" ve "metin2.txt" olarak girilmiştir. Dosya okuma işlemlerinde hangisinin okunacağını kendiniz kod ekranında 80. satırda (*rs = fb.Read("metin.txt");) kendiniz elle de girebilirsiniz.
